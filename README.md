

A static website exploring the Impressionist movement, its artists, and their works. Hosted on GitHub Pages.

## About

This site covers the history, techniques, and legacy of Impressionism alongside detailed artist pages featuring biographies and galleries of selected works. Coverage spans French, American, and international Impressionists.

### Artists included

- **French** — Monet, Renoir, Degas, Morisot, Pissarro, Sisley, Caillebotte
- **American** — Cassatt, Hassam, Robinson, Twachtman, Weir
- **Global** — Sorolla (Spain), Corinth (Germany), Grabar (Russia), Krøyer (Denmark), Streeton (Australia)

## Features

- Sidebar navigation with collapsible artist groups
- Artist pages with portraits, biographies, and selected works
- Lightbox image viewer for full-screen artwork viewing
- Responsive design (sidebar collapses to hamburger menu on mobile)
- JSTOR-inspired academic typography (Source Serif 4 / Source Sans 3)

## Technical details

- Pure static HTML, CSS, and JavaScript — no frameworks or build tools
- All artwork images sourced from Wikimedia Commons (public domain)
- Hash-based client-side routing with browser back/forward support

## Running locally

Open `index.html` in a browser, or serve with any static file server:

```
python3 -m http.server 8000
```
